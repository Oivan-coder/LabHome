# Health ID — расширенный MVP личного кабинета пациента

Это статический кликабельный прототип без backend, БД и настоящей авторизации.

## Как запустить

1. Распакуйте архив.
2. Откройте `index.html` в браузере.
3. Нажмите «Открыть личный кабинет».

## Что добавлено во v2

- Расширенная главная страница с лентой пациента.
- Более развитый раздел лабораторной диагностики.
- Группы исследований: ОАК, биохимия, липидный профиль, коагулограмма, гормоны, воспаление.
- Поиск лабораторных показателей.
- Карточки показателей со sparkline-графиками.
- Большой график динамики с зоной референса.
- Пациентские пояснения к показателям.
- Полная лабораторная история по датам.
- Фильтр «только отклонения».
- Запись к врачу: выбор специальности, врача, даты и времени.
- Создание демо-записи с добавлением в список приемов и ленту событий.
- Раздел приемов.
- Раздел заключений с фильтрами.
- Раздел документов.
- Профиль пациента.
- Drawer быстрых действий.
- Экспорт моковых данных в JSON.

## Где лежат данные

Все данные находятся в `app.js` в объекте `mockData`.

Для боевой версии этот объект можно заменить на backend API:

```http
GET /patients/{mis_card}/summary
GET /patients/{mis_card}/visits
GET /patients/{mis_card}/reports
GET /patients/{mis_card}/documents
GET /patients/{mis_card}/lab-results
POST /appointments/book
```

## Важное ограничение

Это только визуальный и продуктовый MVP. Здесь нет:
- настоящей авторизации;
- проверки прав пациента;
- защиты персональных данных;
- серверной части;
- подключения к МИС/ЛИС;
- медицинской валидации интерпретаций.
